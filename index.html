Ahorcado

Pasos para jugar ahorcado:

  1. Debes escribir una letra que consideres que está presente en la palabra
  2. La letra ingresada no debe ir con tíldes y no contiene la letra 'ñ'
  3. Si la letra está presente, se mostrará en la palabra
  4. Si la letra no está presente, cambiará la imagen para mostrar el progreso
  5. Solo tienes 7 oportunidades
